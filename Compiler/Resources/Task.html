Константы в языке C++

Константы — элементы данных, значения которых известны и в процессе выполнения программы не изменяются.

Синтаксис объявления

Для описания констант в C++ используется ключевое слово const с указанием типа (float или double):

const (float|double) имя_константы = значение;

Примеры констант

1. Целая десятичная константа

Десятичное число без точки (со знаком или без):

const float id = 1111;

2. Вещественная константа с фиксированной точкой

Действительное число с целой и дробной частями:

const double min = -11313.1313;
